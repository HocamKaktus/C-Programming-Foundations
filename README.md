# C Programming Foundations 

This repository contains my early C programming projects and quizzes from Gazi University.

## Contents

### 1. Quiz 1: Basic Logic & Math
A program that takes user input to perform mathematical operations:
- Checks if a given number is prime.
- Calculates the number of digits of a given positive integer.

### 2. Quiz 2: Arrays & Sorting
A data structure exercise focusing on array manipulation:
- Sorts an array of integers (Bubble Sort logic).
- Counts the number of even and odd integers within the array.

### 3. Quiz 3: Pointers & Dynamic Memory Management
This is the most advanced part of the repository, using pointers, string manipulation and malloc/free.

- copyString: Creates a dynamic copy of a string.
- concatenate: Merges two strings into a newly allocated memory space.
- reverseString: Reverses a string using pointer arithmetic.
- Memory Safety: Ensures all allocated memory is properly freed to prevent memory leaks.

## Concepts
- Language: C
- Key Concepts: Pointers, Dynamic Memory Management (`malloc`, `free`), Arrays, Loops, and Functions.

## How to Run
You can compile any of these files using a standard C compiler like GCC or clang.
